# EcoMarket 🌱  

EcoMarket es un proyecto de **e-commerce** diseñado como prototipo web, orientado a la venta de productos sustentables. A través de su **página principal** se presenta una experiencia interactiva donde los usuarios pueden explorar productos, gestionar un carrito de compras y acceder a distintas secciones complementarias del sitio.  

## 🚀 Funcionamiento del proyecto  

La aplicación está construida con **HTML, CSS y JavaScript puro**, organizando sus componentes de manera sencilla para cubrir las funcionalidades esenciales de un comercio electrónico:  

- **Página principal**: contiene la presentación del proyecto y la navegación hacia las secciones clave.  
- **Carrito de compras**: desarrollado con **JavaScript**, permite agregar, visualizar y eliminar productos seleccionados. La lógica implementada actualiza dinámicamente los totales y cantidades.  
- **Formulario de contacto/registro**: utiliza validaciones en **JavaScript** para mejorar la interacción del usuario y asegurar la correcta captura de datos.  
- **Enlace al blog**: redirige a un espacio adicional de contenido relacionado con la temática del proyecto, fomentando la interacción más allá de la compra.  
- **Backoffice**: incluye un prototipo de panel para la administración básica, permitiendo simular la gestión de productos y usuarios.  

## 🛠️ Tecnologías utilizadas  

- **HTML5** → estructura semántica del sitio y definición de las distintas vistas.  
- **CSS3** → estilos personalizados, diseño adaptable y presentación visual acorde a la temática de comercio sustentable.  
- **JavaScript** → lógica de interacción, validaciones de formularios y actualización dinámica del carrito de compras.
